# Kim López Güell — Personal Website

Professional portfolio website showcasing research, projects, and academic contributions.

## Deploying on GitHub Pages

1. Create a new repository on GitHub (e.g., `KimLopezGuell.github.io` for a personal site, or `Website` for a project site).
2. Push these files to the repository.
3. Go to **Settings → Pages** in the repository.
4. Under **Source**, select the branch (e.g., `main`) and root folder (`/`).
5. Your site will be live at `https://kimlopezguell.github.io/` (or `https://kimlopezguell.github.io/Website/`).

## Files

- `index.html` — Main HTML structure
- `style.css` — Complete styling with dark theme design system
- `script.js` — Interactivity (navbar, animations, smooth scrolling)

## Customisation

- **Content**: Edit `index.html` to update text, add publications, or modify project descriptions.
- **Colours**: Modify CSS custom properties in `:root` at the top of `style.css`.
- **Sections**: Each section is self-contained and can be rearranged or removed independently.
